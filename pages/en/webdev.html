Web Development

I had a lot of fun customizing it and set myself the goal of learning how to code it “from scratch.” I completed a few online tutorials to learn Python and realized that I really love programming. So, I decided to enroll in an intensive but comprehensive training program to turn it into my profession.

Once I grasped the syntax, I really enjoyed working with this language, especially after learning about object-oriented programming and the Symfony framework. I'm learning laravel by myself in my free time.
